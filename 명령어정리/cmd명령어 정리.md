## CMD명령어 즉, 명령 프롬프트를 말함. 구글링 귀찮으니..

    윈도우의 중요 사항을 변경하려면 CMD도 관리자 권한으로 실행해주는 것이 좋음
    ※ 모든 명령어에 /? 붙이면 관련 도우말이 나옴 꼭 기억하자
    
 - 배경색 및 글씨 폰트 바꾸기

```
COLOR /? 
```

 - IP주소 확인

```
ipconfig 
```

  - 폴더 이동

```
cd
```

  - 화면 지우기

```
cls
```
